WorkFlow

WorkFlow is a mobile application built with Swift and Firebase that connects homeowners with contractors for job fulfillment. The app features secure authentication, user profiles for both homeowners and contractors, real-time chat for seamless communication, and a bidding system that allows contractors to place and outbid job offers with live status updates. Homeowners can post jobs, review contractor bids, and provide ratings upon completion, while contractors can showcase their skills and build credibility through reviews. WorkFlow demonstrates full-stack mobile development, integrating a responsive iOS interface with a real-time cloud backend. This project was done with three other members of our software engineering class and it is currently not deployed.
